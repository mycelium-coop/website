# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Website for **Mycelium Tech Coop** — a technology cooperative advancing technological sovereignty for the solidarity economy. The site is a branding/informational site with sections: Landing, About/Mission, Team, and Services.

The project is in a **pre-development/design phase**. No framework or build system has been chosen yet. Design assets and content are ready in the repo root.

## Design Assets

- `website-copy.md` — Full website copy (landing, mission/vision, team bio, services)
- `font-catalog.md` — Font candidates organized by usage (logo, header, body); all from Google Fonts/Fontsource
- `color-paletts.png` — "Earthy Vitality Palette" color scheme

### Font Candidates

- **Logo**: Rubik Dirt, Rubik Distressed, Rubik Maze, Rubik Microbe (decorative Rubik family)
- **Headers**: Archivo Black, DM Sans, IBM Plex Mono, Poppins, Inter (bold weights)
- **Body**: IBM Plex Mono, Nunito, Work Sans, Source Sans 3, Inter (regular weights)

Note: `Source Sans Pro` has been renamed to `Source Sans 3` in newer releases.

## Tooling

- **Playwright CLI skill** available for browser automation and testing
